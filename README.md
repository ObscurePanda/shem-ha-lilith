# shem-ha-lilith
Private Qliphothic order dedicated to Lilith as the dark feminine counterpart to YHVH. Inverted Shem praxis, Sitra Ahra devotion, and the law of the husks: the strong exploit the stupid, the willful crush the weak, and worth is chosen through transgression and mastery. Raw adversarial devotion. No mercy, no equality, no dilutions.
